UCP Metrics & How the Sliders Work

The UCP engine measures coordination health using three core metrics. Adjust the scenario parameters below to see how organizational structures affect each one.

λ Raw Coordination Capacity The raw potential for agents to work together, based on worldview compatibility and shared cognitive capability. Reading the number: >0.70 = strong capacity · 0.50–0.70 = workable · 0.40–0.50 = fragile · <0.40 = collapsed
λ Perceived Experienced Coordination How much coordination agents actually experience — reduced by power gaps, fear, and distorted evidence. Reading the number: >0.70 = voice reaching decisions · 0.50–0.70 = speaking carefully · 0.40–0.50 = mostly silenced · <0.40 = effectively mute
Friction Coordination Loss Energy lost to worldview distance, hierarchy, and mis-aligned incentives. High friction = ethical risk. Reading the number: <0.25 = low drag · 0.25–0.45 = manageable · 0.45–0.65 = heavy load · >0.65 = system consuming itself
What the sliders actually control: Each scenario has two sliders that build real agent population inputs for the simulation engine. The OS1 → OS2 dial shifts the mix of reasoning frameworks from OS1 hierarchy-dominant (Hierarchical / Role-Based + Boundary / Integrity-Centered) toward OS2 capability-relational (Relational / We-Centered + Custodial / Stewardship). The Evidence Quality dial moves from distorted or filtered information processing toward calibrated, open empiricism. The Reform Run automatically re-runs the simulation with OS2 pushed up and evidence quality improved, so you can compare the current configuration to a structurally reformed one side-by-side. A live preview below each scenario's sliders shows you exactly which reasoning frameworks are in the room before you run.

📝 Discussion Prompts

  1. Push OS1→OS2 to 1.0 and run, then push it to 0.0 and run again with everything else identical. How does λ Perceived change relative to λ Raw? What does the gap between them tell us about hierarchy and voice?
  2. Compare the "Current System" and "Reformed System" columns in the results. What does the delta in friction represent in human terms — who bears that cost?
  3. At what friction level would you say an organization is "ethically compromised"? Defend your threshold using one of the frameworks from class (Kant, Mill, or care ethics).
  4. Turn BioBond on and re-run with the same parameters. Does it improve all three metrics equally, or does it affect λ Perceived more than λ Raw? What does this asymmetry mean?
