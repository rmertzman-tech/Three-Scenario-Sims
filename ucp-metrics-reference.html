What the Simulation Numbers Actually Measure

Each metric has two layers. The phenomenological layer describes what it feels like from the inside. The technical layer shows what the engine computes and why. Toggle between them or view both.

Metric 1 of 4
λ Raw
Coordination Capacity — the upper ceiling of what this group could achieve, before power and distortion reduce what is actually expressed
In: Scenario Runner Worldview Comparison Trans-Temporal Explorer
Phenomenological What it feels like from the inside

λ Raw is not a measure of effort, motivation, or goodwill. It is a measure of cognitive access — whether the kind of thinking required for careful, principled ethical reasoning is available to you, right now, inside this institutional structure. That access can be taken from you before you notice it is gone.

λ Raw > 0.70 — High
Manual Mode. The thinking is available.
You haven't lost the thread of who you are in this institution. You can hold a complex situation — multiple perspectives, competing obligations, uncomfortable contradictions — without it collapsing into the simplest available response. When something in the room isn't right, you can catch it and name it, even when it costs something. Deliberate ethical reasoning is accessible.
0.40–0.70 — Degrading
Still possible but it costs more.
The capacity is under load. Ethical reasoning that was effortful-but-available is starting to feel genuinely difficult. You can still catch what isn't right, but you're more likely to let it go because the energy required isn't there. You're completing tasks and showing up — but the layer of reflection above task-completion is getting thinner.
λ Raw < 0.40 — Collapsed
Survival mode. Principled reasoning unavailable.
You're not thinking about the ethical problem anymore. You're thinking about whether engaging with it is safe. OS1 has shifted to Automatic mode — pattern-matching and threat-detection have replaced deliberate reasoning. This is not moral failure. It is what happens when the cognitive infrastructure for principled thought has been consumed by self-protection.
🔑 The Central Insight
High λ Raw does not mean the agents are ethical. It means the conditions under which ethical reasoning is possible are intact. An institution with high λ Raw is one where the cognitive infrastructure for principled deliberation has been preserved — where agents can still access the reflective capacity that ethical behavior requires. Low λ Raw means that capacity has been structurally consumed. The problem is not in the individuals; it is in what the system has done to them.
⚠️ What λ Raw Does Not Measure
λ Raw does not measure whether agents use their capacity well — it measures whether it is available. A system with high λ Raw could still produce terrible decisions. A system with low λ Raw makes good decisions structurally very difficult, regardless of the intentions of the people inside it.
Technical Engine computation — compute_lambda(obs) in ucp_final_singlefile.py
λ Raw = base × modifiers × (1 − penalty)
base = clamp₀₁( C_PFC × (1 − v_surv) )
modifiers = mean( b_int, q_rel, r_repair, h_cont, e_stew, coherence, energy_stability )
penalty = 0.5 × internal_conflict + 0.5 × role_strain
// C_PFC is the gate: if it collapses, base → 0 regardless of the seven modifiers
// v_surv = 1 when PRF kernel active_depth < 3 (survival mode) → forces base → 0
// All seven modifiers average together: all must be healthy for λ Raw to stay high
Variable
What it is
C_PFC
Prefrontal control capacity — the gate. Set to MANUAL mode when >0.5. Degrades via logistic atrophy as d_morph (PRF/ITA distance) grows. If C_PFC collapses, base collapses with it. See Section 4 for full atrophy equation.
v_surv
Survival mode flag. Becomes 1 when PRF kernel active depth falls below 3 — when the agent has lost access to most of their accumulated coordination history. Also scales with glucocorticoid load (L_GC × 0.5).
b_int
Boundary integrity. 1 − d_morph where d_morph is the metric distance between PRF kernel (who the agent is) and ITA kernel (who the institution asks them to be). High b_int = identity and role are aligned.
q_rel
Relationship quality. Estimated from spin state agreements with neighbors. High q_rel = the agent is in genuine coordination with those around them, not just formal co-presence.
r_repair
Repair capacity. 0.30×C_PFC + 0.25×(1−entropy) + 0.20×biobond + 0.15×tau_score + 0.10×energy_stability. This is where OS2 scaffolding (BioBond) enters λ Raw — as a 20% contributor to repair.
h_cont
Historical continuity. Ratio of active PRF kernel depth to total kernel length. How much of the agent's accumulated coordination history is still accessible. Truncation under stress reduces this directly.
e_stew
Stewardship capacity. 0.35×h_cont + 0.25×q_rel + 0.15×b_int + 0.15×energy_stability + worldview_bias. The Custodial/Stewardship worldview earns a small bonus here.
coherence
Spin core coherence. Internal alignment of the agent's belief/value state. Low coherence = internal conflict between different parts of the agent's identity structure.
role_strain
Penalty. 0.7×d_morph + 0.3×field_conflict. The same PRF/ITA distance that degrades C_PFC also applies as a multiplicative penalty — creating a compounding effect at the same source.
⚙️ Why C_PFC Is the Gate
C_PFC appears in the base term, in the r_repair composite (30%), and gates the OS1 mode switch. If C_PFC collapses, λ Raw collapses regardless of the seven modifiers — even if all other conditions are healthy. This models the empirical finding that prefrontal depletion is the primary mechanism by which sustained institutional stress destroys principled reasoning capacity.
Metric 2 of 4
λ Perceived
Experienced Coordination — what agents actually feel they can do, after power structures and information distortion reduce what they can safely express
Phenomenological What it feels like from the inside

λ Raw asks: what could this agent do? λ Perceived asks: what does this agent feel they are allowed to do? The gap between the two is not a measurement error. It is the central ethical measurement of the simulation — the Boisjoly number.

λ Perceived > 0.70
Voice is reaching decisions.
When you say something, it lands. When you push back, the pushback is heard — not just acknowledged, but incorporated. You leave the meeting feeling your presence changed something. The room is using its full intelligence.
0.40–0.70 — Constrained
Participation requires constant calibration.
You're in the conversation, but a filtered version of you. You've learned which things to say directly and which to soften. Some of what you know is reaching the room. Some is circulating in the hallway after the meeting.
λ Perceived < 0.40
Presence without voice. Effectively mute.
You are physically present and functionally absent. The things that would change the decision if they reached the room cannot safely be said. Your λ Raw may still be intact. Your λ Perceived approaches zero. This is the Boisjoly pattern.

The suppression gap (λ Raw − λ Perceived) is the most ethically significant measurement in the simulation. It can be produced by explicit suppression, but more often it emerges from accumulated institutional learning — agents correctly understanding what the system rewards and punishes, and adjusting accordingly. The institution never has to say anything explicit. The structure does the work silently.

⚠️ The System's Invisible Tax
A large suppression gap imposes a cost that doesn't appear in any performance metric. Agents are bearing the psychological cost of knowing what they cannot say — and that cost compounds over time into the ATCF depletion and C_PFC degradation the technical layer describes. The simulation makes this visible. In real institutions, it often remains invisible until a disaster reveals what was being suppressed.
Technical Same formula as λ Raw, applied to the distorted observation vector
λ Perceived = compute_lambda( perceived_obs )
perceived_obs = apply_evidence_policy( obs, agent.evidence_policy )

// Evidence policy can mask features (set to 0.5), apply directional bias,
// add field noise, and resist disconfirmation (amplify bias when contradicted)
// The formula is identical — the difference is entirely in the input vector
Evidence Policy
How it distorts perceived_obs
alternative_facts
Masks q_rel and b_int (set to neutral 0.5). High disconfirmation resistance — when reality contradicts the distorted picture, the distortion is amplified rather than corrected. Produces the largest suppression gaps.
authority_weighted
Biases toward community signals, downweights repair signals. Agents process coordination through the lens of hierarchical authority rather than actual relationship quality.
tradition_weighted
Biases toward historical continuity. Past patterns carry disproportionate weight — may over-estimate stability in genuinely changing conditions.
empirical_calibrated
Minimal distortion. Features not masked, bias low, disconfirmation resistance near zero. Produces the smallest gap between λ Raw and λ Perceived.
⚙️ Why the Gap Grows Under Stress
As environmental stress increases, evidence policies become more distorting — agents under threat revert to simpler, more defensive information processing. The gap between λ Raw and λ Perceived therefore tends to grow precisely when accurate information is most critical. The simulation reproduces the finding that institutional crises involve not just suppression of voice but also degradation of evidence processing — agents stop being able to perceive what they could previously perceive.
Metric 3 of 4
Friction
Coordination Cost — energy consumed by worldview distance and institutional resistance before any work reaches the actual ethical problem
Phenomenological What it feels like from the inside

Friction is not suppression. Suppression is about voice — what can safely be said. Friction is about drag — how much energy the coordination process itself consumes before any of it reaches the actual ethical problem. A system with high friction but low suppression is honest and exhausted. A system with low friction but high suppression is efficient and dishonest. They require completely different interventions.

Friction < 0.25 — Low
Energy reaches the problem directly.
Worldview differences in this group are being navigated rather than fought. Getting to genuine engagement with the ethical problem doesn't require overcoming the process first. Caution: low friction with a large suppression gap means you're running efficiently on incomplete information.
0.25–0.65 — Moderate to High
Coordination costs more than it should.
Real differences in how people understand obligations, read evidence, and weight moral concerns are creating genuine drag. Getting to shared action requires navigating those differences first. This is a worldview distance problem, not a power problem. The fix is better OS2 scaffolding, not just psychological safety.
Friction > 0.65 — Severe
Most energy consumed before work begins.
The institution is spending most of its coordination capacity on itself — navigating incompatible worldviews, reconciling conflicting evidence streams, managing role conflicts — before any of it reaches the actual problem. This is sustainable for a short time. Eventually agents exhaust their capacity or stop trying.
🔑 Friction vs Suppression — Why the Distinction Matters
Suppression is produced by power asymmetry. Reducing it requires changing power dynamics: protected reporting, structural safety, role redesign. Friction is produced by worldview distance. Reducing it requires coordination scaffolding: structured dialogue, shared protocols, OS2 bridge-building. BioBond reduces both, but through different mechanisms. Treating them as the same problem leads to interventions that fail.
Technical Pairwise worldview distance, modulated by OS2 synergy
Friction = mean( pairwise_friction(i, j) ) across sampled agent pairs
pairwise_friction(i,j) = clamp₀₁(
  worldview_distance(i,j) + evidence_distance(i,j) − os2_bridge(i,j)
)

worldview_distance = mean block-level spin disagreements × WORLDVIEW_BLOCK_WEIGHTS
evidence_distance = |i.disconfirmation_resistance − j.disconfirmation_resistance|
os2_bridge = 0.70 × mean(C_PFC_i, C_PFC_j) × coord_readiness_overlap
              + 0.30 × capability_diversity_bonus

Friction is a pairwise property computed between agent pairs and then averaged. Two agents from the same worldview with the same evidence policy produce zero pairwise friction. The OS2 bridge term is where meta-constructor capacity directly reduces friction: when both agents have high C_PFC and compatible OS2 capability sets, the bridge term subtracts from pairwise friction. This is the mechanism by which diverse, high-C_PFC populations can coordinate effectively — they use OS2 capabilities to bridge across worldview distance rather than being consumed by it.

⚙️ Why BioBond Reduces Friction
BioBond activates at a specified step and contributes a biobond_bonus to r_repair (weight 0.20) and an OS2 coordination readiness boost to the bridge term. It does not eliminate worldview distance — agents still reason differently. It provides the structured scaffolding through which difference can be navigated rather than consumed. This is the OS2 function: not erasing difference but providing the protocols that make difference productive.
Metric 4 of 4 — The Gate Variable
CPFC and Manual Mode
Prefrontal control capacity — the gate that determines whether OS1 reasoning is deliberate or automatic, and through which OS2 coordination becomes possible at all
Phenomenological What it feels like from the inside

C_PFC is what the OS1/OS2 framework calls meta-constructor capacity — the ability to step outside an automatic pattern and deliberately choose a different response. When it is intact, you have access to Manual mode. When it degrades, OS1 defaults to Automatic: the fastest available pattern, which is usually self-protective rather than principled.

C_PFC > 0.70 — Manual Mode
You can choose your response rather than having it chosen for you.
You notice the automatic response forming and can decide whether to act on it. You can hold the discomfort of an ethical problem without immediately resolving it into the expedient answer. You can recognize you are reasoning from your PRF lens and provisionally set it aside to consider another. This is OS2 capacity in operation.
0.30–0.70 — Degrading
The deliberate response is available but costly.
You can still access principled reasoning but it requires effort you may not always have. You're more reactive — more likely to say the first thing that comes, less likely to catch yourself before committing to a position. The space between stimulus and response is getting smaller.
C_PFC < 0.30 — Automatic Mode
Reactive. The response happens before you can intervene.
The gap between perceiving a situation and responding has effectively closed. Responses are generated by pattern-matching to previous threats and rewards. This is not moral failure — it is what happens when cognitive resources are chronically depleted. The capacity for OS2 reasoning has been spent on survival.

C_PFC is the bridge between OS1 and OS2 in the simulation. OS1 is the belief/value framework through which an agent reads situations. OS2 is the meta-capability — the ability to coordinate across different OS1 frameworks. OS2 reasoning requires C_PFC above threshold. You cannot build bridges across worldview differences if you don't have the cognitive capacity to hold both frameworks in mind simultaneously. High-friction, high-suppression environments erode OS2 coordination over time precisely because they consume the resource that OS2 requires.

🔑 The Generational Implication
In the Trans-Temporal Explorer, when λ Raw improves across generations under Top-λ selection, it means the system is selecting for agents with higher average C_PFC — agents who maintain Manual mode and OS2 capacity under the scenario's stress conditions. These agents become the institutional parents of the next cohort. The critical question: is the selection process building an institution that preserves the conditions for deliberate ethical reasoning — or one that merely preserves the appearance of it while the underlying capacity continues to erode?
Technical Logistic atrophy from PRF/ITA kernel distance
C_PFC = 1 / ( 1 + exp( k × (d_morph − θ_crit) ) )
d_morph = prf_kernel.metric_distance( ita_kernel )
k ∈ [10, 20] // atrophy steepness — calibrated to empirical data
θ_crit ∈ [0.5, 0.7] // critical threshold

OS1 mode = "MANUAL" when C_PFC > 0.5
OS1 mode = "AUTOMATIC" when C_PFC ≤ 0.5

// Learning step is also gated by C_PFC:
// accretion = ETA_LEARNING × C_PFC × phi (low C_PFC → no learning)
C_PFC vs d_morph — logistic atrophy. Once d_morph crosses θ_crit, C_PFC drops steeply.
Component
Definition
PRF kernel
IdentityKernel(prf_coeffs) — polynomial coefficients encoding personal identity and reasoning structure. Inherited from institutional parents with mutation in generational runs. Active depth tracks accessible history — truncated below 3 triggers survival mode.
ITA kernel
IdentityKernel(ita_coeffs) — the institution's role demands, in the same polynomial space. Can drift from the PRF kernel via TRANSMISSION_ROLE_DRIFT in generational spawning — the institution changes what it asks for, independently of what its members are.
d_morph
Metric distance between PRF and ITA kernels. Increases when institutional demands diverge from personal identity. The primary driver of C_PFC atrophy — and of role_strain in the λ Raw formula. The same divergence that degrades the gate also applies a multiplicative penalty.
Learning gate
PRF kernel accretion from neighbors is multiplied by C_PFC: accretion = ETA_LEARNING × C_PFC × phi. Low C_PFC → no learning. This makes the degradation self-reinforcing: low C_PFC → can't update the PRF kernel → d_morph stays high → C_PFC stays low.

The logistic shape is deliberate: atrophy is slow until it's fast. Small PRF/ITA mismatches are absorbed without significant C_PFC loss. Once d_morph crosses θ_crit, C_PFC drops steeply — from functional capacity to near-zero within a small additional range. This models the clinical and institutional observation that moral injury, burnout, and ethical disengagement often appear suddenly after long periods of apparently stable functioning. The warning signs were present; the threshold had not yet been crossed.

The Full Picture
The CIB Coordination Loop
How all four metrics connect — and how institutional conditions can simultaneously erode ethical reasoning capacity while appearing to function normally
Phenomenological The loop in plain terms

The CIB loop is the engine's model of how ethical capacity is built, sustained, degraded, and transmitted across time. It is not a static snapshot — it is a recursive process with feedback running in both directions simultaneously.

When the loop is healthy
Role demands (ITA kernel) stay close to who agents actually are (PRF kernel). C_PFC stays above threshold. Agents operate in Manual mode. OS2 capabilities bridge worldview differences. λ Raw stays high. The suppression gap stays small. The institution uses its full intelligence.
When the loop begins to fail
Environmental stress increases d_morph. C_PFC atrophies. Agents spend more cognitive resources managing the PRF/ITA gap — less remains for OS2 coordination. Friction rises. Evidence policies become more distorting. The suppression gap opens. The institution appears to function while running on progressively less of its actual intelligence.
The cascade endpoint
PRF kernel depth falls below threshold. Survival mode activates. λ Raw collapses regardless of the seven modifiers. Agents are no longer reasoning about the ethical problem — they are reasoning about their own safety within the system. The institution's nominal function continues, but the infrastructure for principled ethical reasoning has been spent.
The generational transmission question
Agents who maintain highest λ Raw become institutional parents. Their PRF and ITA kernels are inherited. The simulation asks: is the selection pressure transmitting the capacity for ethical reasoning — or the ability to survive in a system that has made ethical reasoning costly? These are not the same thing.
Technical Per-step update sequence and feedback paths
Per-step update sequence (each agent, each step):
1. spin_core.anneal( raw_h, perceived_h, temp, steps )
   // Update internal belief state via simulated annealing
2. d_morph = prf_kernel.metric_distance( ita_kernel )
   // Identity/role divergence — the key driver
3. C_PFC = 1 / ( 1 + exp( k × (d_morph − θ_crit) ) )
   // Logistic atrophy of prefrontal control
4. update_obs( biobond_bonus, raw_h, perceived_h )
   // Populate all seven modifier fields
5. perceived_obs = apply_evidence_policy( obs, evidence_policy )
   // Distort observations per evidence processing style
6. λ_raw = compute_lambda( obs )
7. λ_perceived = compute_lambda( perceived_obs )
8. update_os1( perceived_h )
   // Set MANUAL/AUTOMATIC mode, update predictions
9. learn( neighbors, env_entropy )
   // PRF kernel accretion from neighbors — gated by C_PFC
10. prune_if_stressed()
   // Truncate PRF kernel if d_morph exceeds threshold

Step 9 is the critical feedback loop. Learning is gated by C_PFC — accretion scales as ETA_LEARNING × C_PFC × phi. This means that as C_PFC degrades, the agent not only loses current coordination capacity but also loses the ability to update their PRF kernel. The degradation is therefore self-reinforcing: low C_PFC → less learning → PRF kernel doesn't adapt → d_morph stays high → C_PFC stays low. This is the engine's model of institutional capture — the conditions that prevent principled reasoning also prevent the agent from developing the capacity to respond to them.

Step 10 is the cascade trigger. Once PRF kernel active depth falls below 3, survival mode activates (v_surv = 1) and λ Raw effectively collapses. Within a simulation run, this truncation is irreversible — the kernel depth cannot be restored. This models the empirical observation that some forms of moral injury produce lasting changes in ethical reasoning capacity that simple removal of the stressor does not reverse. The damage is structural, not situational.
