Framework Profiles — Two Lenses on the Same Frameworks

Two charts read each framework through different lenses. Chart 1 (UCP Coordination Profile) shows how the framework weights coordination signals in the simulation engine — what it amplifies and what it discounts when agents are trying to work together. Chart 2 (Moral Foundations Profile) shows the psychological architecture of the framework using Haidt et al.'s five moral foundations — the emotional and intuitive building blocks from which moral reasoning is constructed. The same framework looks different through each lens, and both lenses are needed to understand why two frameworks produce the simulation numbers they do.

📝 Discussion Prompts

  1. Look at the radar overlay. Which axes diverge most between the two frameworks? What would a disagreement look like in practice when agents with these profiles are in the same room?
  2. Compare λ Perceived across the three columns. Does mixing the frameworks increase or decrease experienced coordination? Why might that surprise you?
  3. Read the agent voices side by side. Could you identify which framework is speaking if the labels were removed? What language cues give each framework away?
  4. Which framework produced lower friction in this scenario? Does that mean it is the more ethical framework — or just the more coordinated one? Is there a difference?
